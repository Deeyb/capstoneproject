What is Java?
Java is an object-oriented programming language with a built-in
application programming interface (API) that can handle graphics and
user interfaces and that can be used to create applications or applets.
Java Technology consists of:
- Java Language - Java Platform – Java Virtual Machine
- Java Tools – Java Compiler/Debugger

Java, Web, and Beyond
• Java can be used to develop standalone applications.
• Java can be used to develop applications running from a browser.
• Java can also be used to develop applications for hand-held devices.
• Java can be used to develop applications for Web servers.

History of Java
Java was created in 1991 by James Gosling et al. of Sun Microsystems.
Initially called Oak, in honor of the tree outside Gosling's window, its
name was changed to Java because there was already a language called
Oak.
The original motivation for Java was the need for platform independent
language that could be embedded in various consumer electronic
products like toasters and refrigerators. One of the first projects
developed using Java was a personal hand-held remote control named
Star 7.
At about the same time, the World Wide Web and the Internet were
gaining popularity. Gosling et. al. realized that Java could be used for
Internet programming.

Using Scanner Class
The Scanner class is a class that provides methods for parsing and processing input from various sources such as keyboard input,
files, and strings. The Scanner class is part of the Java.util package and is used for reading input of various data types from a
variety of sources.
The Scanner class provides a range of methods to read different types of data such as integers, floats, doubles, and strings. The
class uses regular expressions to define the pattern of the input, which is then used to match and read the input. The class also
has methods to skip over input that is not required, and to detect the end of the input. Here are some of the common methods of
the Scanner class:
• next(): This method reads the next token of input as a string.
• nextLine(): This method reads the next line of input as a string.
• nextInt(): This method reads the next token of input as an integer.
• nextDouble(): This method reads the next token of input as a double.
• nextBoolean(): This method reads the next token of input as a boolean.
• hasNext(): This method returns true if there is another token in the input stream.
• hasNextLine(): This method returns true if there is another line of input in the input stream.
• useDelimiter(String regex): This method sets the delimiter pattern for the Scanner.
• close(): This method closes the Scanner object.

dialogs, input dialogs, option dialogs, and confirmation dialogs.
